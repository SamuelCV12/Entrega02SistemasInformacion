# 🚀 Entrega 02: Sistemas de Información - Quipux S.A.S.

## 🛠️ Especificación de la Solución Propuesta

La solución desarrollada para **Quipux S.A.S.** consiste en un ecosistema automatizado de **extracción, consolidación y persistencia de datos**. El objetivo principal es eliminar la carga operativa manual, permitiendo que la información crítica esté disponible de forma estructurada para la toma de decisiones estratégicas.

---

### 📑 Descripción General del Sistema

El sistema ha sido diseñado bajo una arquitectura de **procesamiento autónomo**, operando mediante un **orquestador central** que garantiza la sincronización de las siguientes etapas:

1.  **Extracción (Harvesting):** Captura automatizada de datos desde fuentes heterogéneas.
2.  **Estructuración:** Transformación de datos crudos en esquemas organizados y normalizados.
3.  **Almacenamiento:** Persistencia en un repositorio centralizado para garantizar su disponibilidad inmediata.

> [!IMPORTANT]
> **Alineación Estratégica:** Esta solución interviene directamente en el proceso de **monitoreo de indicadores y rendimiento de plataformas digitales**, identificado previamente como el nodo crítico de la operación.

---

### ⚙️ Funcionamiento y Experiencia de Usuario

A diferencia de las herramientas convencionales de carga manual, nuestra propuesta prioriza la **autonomía del backend**:

* **Ejecución Desatendida:** El usuario final no requiere interactuar con el sistema durante las fases de extracción o procesamiento.
* **Consumo bajo demanda:** La interacción se traslada al resultado final, donde el usuario accede a información ya procesada y lista para su análisis.
* **Salida:** El sistema está capacitado para exportar la información de múltiples plataformas(Google Analitycs, Google Sheet, SurveyMonkey y Gmail) en:
    * 📉 **Plantillas de PowerPoint:** Para reportes ejecutivos automatizados.

---

### 🧩 Componentes del Core Técnico

| Componente | Responsabilidad |
| :--- | :--- |
| **Orquestador Central** | Coordinación de flujos de trabajo y gestión de tiempos de ejecución. |
| **Extractor Engine** | Interfaz con plataformas digitales para la obtención de métricas. |
| **Data Warehouse / Persistencia** | Almacenamiento seguro que garantiza la integridad de la data 24/7. |
